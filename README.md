## 기능

- javascript Slick 라이브러리를 사용하여 슬라이드 효과를 주었습니다.
- 새로운 영화 탭에서 "로그 원: 스타워즈 스토리"를 클릭시 팝업으로 Youtube를 통해 예고편을 보여주게 제작하였습니다.
